# Frontend Angular

Interfaz de Beautiful Skin. Consume los servicios REST mediante el API Gateway
en `http://localhost:8050`.

```bash
npm install
npm start
```

Abrir `http://localhost:4200`.

Para validar la compilación:

```bash
npm run build
```

Este frontend no incluye una configuración de pruebas unitarias. Las pruebas
automatizadas del proyecto están en los servicios Spring Boot.
